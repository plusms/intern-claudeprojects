コマンド詳細（CV記事）

/cv-update-urls 毎週実行 ・ 競合URL取得・L列書き込み
  • 1対象サイトを選択する
  • 2スプシを読み込む（記事名・メインKW・前回L列）
  • 3全記事のメインKWでWebSearchを並列実行 → 上位5URL取得・公式/アフィリ仕分け → 前週差分チェック → インターン確認
  • 4L列に書き込む（上書き）・ログをmdに追記
/cv-update-analyze W2/W4実行 ・ 競合+AIO/関連KW一括分析 → N列・O列書き込み
  • 1対象サイトを選択する
  • 2スプシを読み込む（F〜K列・L列）。F〜K列が未入力の記事があれば入力を促す
  • 3全記事を一括分析（L列をWebFetch・F〜K列を読んでギャップ分析）
  • 4結果を一覧提示 → 確認 → N列・O列に上書き・ログをmdに追記
  • 5Slack通知テキストを出力する
/cv-update-heatmap W3実行 ・ ヒートマップ分析 → P列書き込み
  • 1対象サイトを選択する
  • 2スプシを読み込む（記事一覧取得）
  • 3記事ごとにClarityスクショ3枚（クリック・スクロール・アテンション）を受け取り → 即分析
  • 4全記事の結果を一覧提示 → 確認 → P列に上書き・ログをmdに追記
  • 5Slack通知テキストを出力する
/cv-update-edit 社員承認後 随時実行 ・ HTML差分出力・WP実装
  • 1対象サイトを選択する
  • 2スプシを読み込む（Q列が入力済み・R列が空の記事を対象）
  • 3採用方針を確認して提示する
  • 4WPのHTMLを貼り付けてもらう
  • 5差分HTMLを出力する
  • 6WP実装完了後、R列（実装完了日）に書き込む
