インターン 初回セットアップガイド

Claude Codeのインストールから作業開始までの手順

1. Claude Codeとは何か

Claude Codeは、AIアシスタント（Claude）をターミナル・VSCode上で使えるツールです。

インターンの作業（記事更新・順位施策）は、このClaude Codeを使ってコマンドを呼び出すことで進めます。Claudeがスプレッドシートの読み書き・競合分析・HTML出力まで自動でやってくれるので、インターンはデータを入力してClaude に指示するだけで作業が完結します。

2. セットアップ手順

  1. Claude Codeをインストールする

    以下のURLからClaude Codeデスクトップアプリをダウンロードしてインストールしてください。

    インストール後、Anthropicアカウントでサインインしてください（アカウント情報は別途共有します）。

  2. リポジトリをクローンする

    ターミナル（Windowsならコマンドプロンプト or PowerShell）を開いて以下を実行してください。

    git clone https://github.com/plusms/intern-claudeprojects.git

    デスクトップなど、わかりやすい場所で実行してください。intern-claudeprojectsというフォルダが作成されます。

  3. Claude CodeでフォルダをProjectとして開く

    Claude Codeを起動し、「Add Project」から先ほどクローンしたintern-claudeprojectsフォルダを選択してください。

    これでコマンド（/knowhow-updateなど）が使える状態になります。

3. 使えるコマンド一覧

コマンド 用途 タイミング
/knowhow-update ノウハウ記事の更新 毎出勤時
/cv-update-analyze CV記事の分析・改修方針案の作成 担当記事が来たとき
/cv-update-edit 社員承認後のHTML改修・WP実装 社員からSlack通知を受けたとき
/rank-local 地域記事の順位施策 担当記事が来たとき
/sync コマンドを最新版に更新 「更新しました」と言われたとき

4. コマンドの最新版に更新する方法

担当者がコマンドを更新したときはSlackで連絡が来ます。その際は以下の手順で最新版に更新してください。

  1. Claude Codeを開く
  2. チャットに /sync と入力する

    「GitHubから最新版を手元に反映します。よいですか？」と表示されるので「はい」と答えてください。

  3. 「最新版に更新しました。」と表示されたら完了

5. 困ったときは

Claudeが止まった・エラーが出た・どうすればいいかわからないときは、そのままClaudeに状況を伝えてください。多くの場合、Claudeが次にすべきことを教えてくれます。

それでも解決しない場合は担当者にSlackで連絡してください。
