# /cv-update-edit

## STEP 1：対象サイトを確認する

以下から対象サイトを選んでもらう。

```
どのサイトのCV記事を実装しますか？
honbe / gyn / asami / ginza / smc
```

---

## STEP 2：実装待ち記事を確認する

維持記事管理シート（ID：`1yr1mz3nQmk2YW2qkjlP0wqZyzdK5ReECz7TdsD9KJfw`）の `[サイト]-CV` タブを読み込む。

**対象条件**：Q列（社員対応番号）が入力済みかつR列（実装完了日）が空

対象記事を一覧で提示する。確認を得てから次へ進む。

```
実装待ちの記事

■ [記事名]
  採用番号：[Q列の番号]
  採用された方針：[番号に対応するN列またはO列またはP列の内容]

この記事を進めましょう。
```

---

## STEP 3：HTMLを受け取る

```
対象記事のHTMLをWordPressの「テキスト」タブから全文コピーして貼り付けてください。
```

受け取ったHTMLのタグ構造・class名・文体を読み取る。

実施前に以下を読む。
- `knowledge/block-rules.md`
- `knowledge/guidelines.md`（2章：見出し構成のみ）

---

## STEP 4：HTML差分を出力する

採用方針に沿った差分HTMLを出力する。
元のHTMLのタグ構造・class名・文体に合わせる。
WPの「テキスト」タブに貼り付けて実装できる状態で出す。

---

## STEP 5：スプシを更新する（WP実装完了後）

WP実装が完了したら報告してもらう。報告を受けてからR列に書き込む。

書込前チェック → [R列: 実装完了日] | [今日の日付（YYYY-MM-DD）] | [関連セル矛盾なし?]

承認を得てから該当行のR列に書き込む。
